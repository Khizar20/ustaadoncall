## üé® Design Elements & Features

### Animations
- **Number Tickers**: Animated counters that count up when scrolled into view
- **Scroll Reveal**: Testimonials fade in as user scrolls
- **Hover Effects**: Cards lift and scale slightly on hover
- **Arrow Animation**: Continuous horizontal movement on CTA button

### Color Scheme
- **Primary Color**: Orange (`hsl(22_65%_45%)` / `#CC6E37`)
- **Background**: Light background with card sections
- **Text**: Foreground and muted foreground colors

### Typography
- **Headings**: Light font weight, heading font family
- **Body**: Regular text with muted variants for descriptions
- **Responsive**: Scales from mobile (text-4xl) to desktop (text-6xl)

### Layout
- **Container**: Max-width container with responsive padding
- **Grid System**: Responsive grids (1 column mobile, 3 columns desktop)
- **Spacing**: Consistent vertical padding between sections

### Interactive Elements
- **Buttons**: Rounded full style, various variants (primary, outline, ghost)
- **Links**: All CTAs link to relevant pages (`/services`, `/become-provider`)
- **Cards**: Hover effects with shadow transitions
